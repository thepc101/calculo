Code Examples

Basic Embed

<script>window.CALCULO_API_KEY = 'demo';</script>
<script src="https://calculo-fawn.vercel.app/embed.js"></script>
<div data-calculator="demo_basic"></div>

With Theme Override

<div data-calculator="demo_scientific"
  data-theme="cyberpunk"
  data-primary="#f0abfc">
</div>

Fixed Position (Bottom-Right)

<div data-calculator="demo_basic"
  data-position="fixed"
  data-fixed-bottom="20px"
  data-fixed-right="20px">
</div>

Custom Size

<div data-calculator="demo_scientific"
  data-width="400px"
  data-height="600px"
  data-theme="ocean">
</div>

All Configuration Attributes

data-calculator Calculator ID (required)
data-theme Theme: dark, light, oled, cyberpunk, ocean, forest, sunset, aurora, monochrome, coffee, retro, glass, neumorphism, minimal, corporate, high-contrast
data-type Calculator type: basic, scientific
data-primary Primary color override: #ff0000, rgb(255,0,0), etc.
data-width Widget width (default: 340px)
data-height Widget height (default: auto)
data-position Position: inline (default), floating, fixed
data-fixed-bottom Fixed bottom offset (e.g. 20px)
data-fixed-right Fixed right offset (e.g. 20px)

Setup

Before loading embed.js, set your API key:

<script>window.CALCULO_API_KEY = 'demo';</script>

Use 'demo' for demo configs, or your calc_live_* key for custom calculators.
